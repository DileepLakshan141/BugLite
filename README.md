# BugLite

**BugLite** is a lightweight, deployable bug tracking web application built with **Next.js**, designed to help software teams efficiently track, manage, and resolve issues in their projects. It's portfolio-ready, featuring a modern tech stack and scalable architecture suitable for real-world applications.

The app emphasizes simplicity, speed, and a clean UI while maintaining the core features expected in industrial bug trackers.
